# Flutter Container Widget Demo

## Widget Description

The Container widget is one of Flutter's most commonly used layout widgets. It is used to hold, position, and style other widgets on the screen.

## Real-World Use Case

This project demonstrates a Student Profile Card. The Container widget is used to create a card that displays a student's profile information in a clean and organized way.

## Run Instructions

1. Clone the repository.
2. Open the project in VS Code.
3. Run:

flutter pub get

4. Run:

flutter run

## Three Properties Demonstrated

### 1. width

Controls the horizontal size of the Container.

### 2. padding

Adds spacing between the Container border and its child widgets.

### 3. decoration

Allows customization such as background color, borders, and rounded corners.
